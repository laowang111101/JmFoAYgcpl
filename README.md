# 停车场管理系统 java929

## 项目概述


## 技术栈

- **开发工具**：Idea/Eclipse
- **数据库**：MySQL
- **依赖管理**：Maven
- **前端框架**：Vue.js、ElementUI、ECharts
- **后端框架**：SpringBoot

## 功能模块

### 用户角色

- 普通用户
- 超级管理员

### 主要功能

- **个人信息管理**：用户可管理个人资料。
- **车位管理**：管理员可以对车位进行管理。
- **费用管理**：系统自动计算停车费用。
- **收费信息**：记录和管理车辆的收费信息。
- **用户登录注册**：提供用户注册和登录功能。
- **系统管理**：管理员进行系统设置和维护。
- **车位信息管理**：维护车位状态和信息。
- **车辆信息管理**：管理车辆的基本信息。
- **车辆进出管理**：记录和管理车辆的进出记录。

## 系统角色

系统分为以下两种角色：

- **普通用户**：可以进行个人信息管理、登录注册、查看费用等操作。
- **超级管理员**：拥有系统管理的权限，可以管理车位、费用、用户、车辆信息以及收费信息。

## 运行环境

- **后端**：Java 1.8 或以上版本，SpringBoot 2.x
- **数据库**：MySQL 5.7 或以上版本
- **前端**：现代浏览器

## 目录结构

```
- src/
  - main/
    - java/
    - resources/
    - webapp/
      - static/
      - templates/
- pom.xml
```

## 部署步骤

1. 配置数据库。
2. 通过Maven构建项目并生成可执行的JAR包。
3. 部署前端静态资源到服务器指定目录。
4. 启动后端服务。

## 核心亮点

- **高效稳定**：基于成熟的SpringBoot和Vue技术栈开发，保证了系统的稳定性和高效性。
- **易于维护**：模块化设计，使得系统易于维护和扩展。
- **灵活配置**：支持多种自定义配置，满足不同停车场需求。
